# CollegeName - Simple College Website

This repository contains a simple, single-file responsive website template for a college or educational institution. It's built using pure HTML and styled with Tailwind CSS via CDN, making it easy to deploy and manage.

## Features

*   **Responsive Design**: Optimized for various screen sizes, from mobile to desktop.
*   **Modern UI**: Clean and contemporary design powered by Tailwind CSS.
*   **Single-File**: All HTML, CSS (via CDN), and JavaScript are contained within `index.html` for easy deployment.
*   **Sectioned Content**: Includes typical college website sections like Hero, About, Programs, Admissions CTA, and Contact.
*   **Interactive Mobile Menu**: A basic JavaScript toggle for mobile navigation.
*   **Colored Background**: Utilizes gradients and thoughtful color choices to achieve a professional yet vibrant "college background" aesthetic.

## Technologies Used

*   HTML5
*   Tailwind CSS (CDN)
*   Vanilla JavaScript (for mobile menu)

## How to Use

1.  **Clone the repository** (if this were a repo) or simply **save the `index.html` file** to your local machine.
2.  **Open `index.html`** in your web browser. That's it!

### Customization

*   **Content**: Edit the text within `index.html` to reflect your college's information.
*   **Colors/Styling**: Modify the Tailwind CSS classes directly in `index.html` to change colors, fonts, spacing, etc. The Tailwind CDN includes all default utility classes.
*   **Images/Assets**: For a real project, you'd link to external image assets. For this simple template, text and SVG icons are used.
